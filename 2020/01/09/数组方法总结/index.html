数组方法总结

Posted on 2020-01-09

数组方法总结

push()

可以接收任意数量的参数，把它们逐个添加到数组末尾，并返回修改后数组的长度。 

arr = [1,2,3]
arr.push(4,5,6)
console.log(arr)

pop()

数组末尾移除最后一项，减少数组的 length 值，然后返回移除的项。

arr = [1,2,3]
var item = arr.pop();
console.log(item)//6

shift()

删除原数组第一项，并返回删除元素的值；如果数组为空则返回undefined 。 

 arr = [1,2,3]
var item = arr.shift();
console.log(item)//1

unshift()

将参数添加到原数组开头，并返回数组的长度 。

arr = [1, 2, 3]
var item = arr.unshift(4, 5);
console.log(item)//5
console.log(arr) //[4,5,1,2,3]

splice()

删除

可以删除任意数量的项，只需指定 2 个参数:要删除的第一项的位置和要删除的项数。 例如，splice(0,2)会删除数组中的前两项。

var arr = [1,2,3,4,5,6,7,8,9]
 console.log(arr.splice(1,2))// 返回[2,3]
 console.log(arr)//[1,4,5,6,7,8,9]

插入

可以向指定位置插入任意数量的项，只需提供 3 个参数:起始位置、0(要删除的项数) 和要插入的项。如果要插入多个项，可以再传入第四、第五，以至任意多个项。例如， splice(2,0,4,6)会从当前数组的位置 2 开始插入4，6

var arr = [1, 2, 3, 4, 5, 6, 7, 8]
 arr.splice(2, 0, 4, 6) //返回空数组
 console.log(arr) //[1, 2, 4, 6, 3, 4, 5, 6, 7, 8]

替换

替换:可以向指定位置插入任意数量的项，且同时删除任意数量的项，只需指定 3 个参数:起始位置、要删除的项数和要插入的任意数量的项。插入的项数不必与删除的项数相等。

 var arr = [1, 2, 3, 4, 5, 6, 7, 8]
arr.splice(0，4, 10，11) //
console.log(arr) //[10,11,5,6,7,8]

join() 拼接

var arr = [1,2,3,4,5]
console.log(arr.join('-))// 1-2-3-4-5

##迭代方法

every()

对数组中的每一项运行给定函数，如果该函数对每一项都返回 true，则返回 true。

var arr = [1,2,3]
var arr2 = arr.every(function(item,index,array){
         return item > 1
})
console.log(arr2) //false

filter()

对数组中的每一项运行给定函数，返回该函数会返回 true 的项组成的数组。

var arr = [1,2,3,4,5,6]
    var arr2 = arr.filter(function(item,index,array){
             return item > 3
    })
    console.log(arr2) //[4,5,6]

forEach()

对数组中的每一项运行给定函数。这个方法没有返回值。

arr.forEach(function(item, index, array){
 });

map()

对数组中的每一项运行给定函数，返回每次函数调用的结果组成的数组。

arr.map(function(item, index, array){
  retrun item >3
});

some()

对数组中的每一项运行给定函数，如果该函数对任一项返回 true，则返回 true

var arr = [1,2,3]
   var arr2 = arr.every(function(item,index,array){
            return item > 1
   })
   console.log(arr2) //false

indexOf()、lastIndexOf()

indexOf()和 lastIndexOf()。这两个方法都接收 两个参数:要查找的项和(可选的)表示查找起点位置的索引。其中，indexOf()方法从数组的开头(位 置 0)开始向后查找，lastIndexOf()方法则从数组的末尾开始向前查找。

var numbers = [1,2,3,4,5,4,3,2,1]; 
 alert(numbers.indexOf(4)); //3
 alert(numbers.lastIndexOf(4)) //5

reduce() 、reduceRight()

这两个方法都会实现迭代数组的所有项，然后构建一个最终返回的值。reduce()方法从数组的第一项开始，逐个遍历到最后。而 reduceRight()则从数组的最后一项开始，向前遍历到第一项。

这两个方法都接收两个参数：一个在每一项上调用的函数和（可选的）作为归并基础的初始值。

传给 reduce()和 reduceRight()的函数接收 4 个参数：前一个值、当前值、项的索引和数组对象。这个函数返回的任何值都会作为第一个参数自动传给下一项。第一次迭代发生在数组的第二项上，因此第一个参数是数组的第一项，第二个参数就是数组的第二项。 

var values = [1,2,3,4,5];
var sum = values.reduceRight(function(prev,cur,index, array){
return prev + cur;
},10);
console.log(sum); //25

以上就是常用的数组方法。
